# Screenshots for README & social posts

Add your PNG/JPEG files here using the **same filenames** referenced in the root [README.md](../../README.md) **Screenshots** section so images render on GitHub.

Suggested captures:

| File | What to capture |
|------|-----------------|
| `01-app-chat-ui.png` | Chat UI in the browser |
| `02-github-actions.png` | Successful workflow run (deploy / build) |
| `03-argocd-synced.png` | Argo CD application **Synced** + revision |
| `04-kubectl-or-helm.png` | `kubectl get pods` or `helm list` (optional) |
| `05-architecture.png` | Simple diagram (draw.io, Excalidraw, or napkin sketch) |

After adding files, commit and push — GitHub will show them in the README automatically.
